อธิบายความแตกต่างระหว่าง Inline, Internal และ External CSS

การใช้ CSS (Cascading Style Sheets) เพื่อกำหนดรูปแบบการแสดงผลของ HTML เป็นสิ่งที่สำคัญในการออกแบบเว็บเพจให้สวยงามและใช้งานง่าย โดยสามารถใช้งานได้หลายวิธี ซึ่งหลักๆ แล้วจะมี 3 รูปแบบหลักๆ คือ Inline CSS, Internal CSS, และ External CSS โดยทั้งสามแบบนี้มีวิธีการใช้งาน

1. Inline CSS (CSS ภายในแท็ก HTML)Inline CSS คือการกำหนดสไตล์โดยตรงในแต่ละแท็ก HTML ผ่านแอตทริบิวต์ style ที่อยู่ภายในแท็กนั้นๆ วิธีนี้จะช่วยให้คุณสามารถปรับแต่งรูปแบบเฉพาะเจาะจงขององค์ประกอบแต่ละอันได้ทันทีโดยไม่ต้องใช้ไฟล์แยกต่างหาก

2. Internal CSS (CSS ภายในไฟล์ HTML)Internal CSS คือการเขียน CSS ภายในแท็ก style ที่อยู่ในส่วน head ของไฟล์ HTML ซึ่งจะช่วยให้คุณสามารถกำหนดสไตล์สำหรับองค์ประกอบต่างๆ ของหน้าเว็บได้อย่างเป็นระเบียบ โดยไม่ต้องแยกไฟล์ CSS ออกไป ตัวอย่างการใช้ Internal CSS

3. External CSS (CSS ภายนอกไฟล์ HTML) External CSS คือการแยกไฟล์ CSS ออกมาเป็นไฟล์ที่มีนามสกุล .css และใช้แท็ก link ในส่วน head ของไฟล์ HTML เพื่อเชื่อมโยงไฟล์ CSS เข้ากับ HTML วิธีนี้ถือเป็นการใช้ที่ดีที่สุดหากต้องการให้เว็บไซต์ของคุณมีสไตล์ที่สามารถใช้ซ้ำได้หลายๆ หน้า ตัวอย่างการใช้ External CSS

แหล่งที่มา

การใช้งาน div และ span แตกต่างกันอย่างไร?

div: ใช้สำหรับการจัดกลุ่มเนื้อหาหรือส่วนต่างๆ ของหน้าเว็บ เป็น block-level element (จะขึ้นบรรทัดใหม่).

span: ใช้สำหรับจัดการเนื้อหาภายในบรรทัดเดียวกัน เป็น inline element (ไม่ขึ้นบรรทัดใหม่).

bootstrap ทำหน้าที่อะไรและมีคำสั่งใดบ้างที่สำคัญ?

ทำให้การออกแบบเว็บแบบตอบสนองเป็นจริงได้ ช่วยให้หน้าเว็บหรือแอปสามารถตรวจจับขนาดและทิศทางของหน้าจอของผู้เยี่ยมชมได้ และปรับการแสดงผลให้เหมาะสมโดยอัตโนมัติ แนวทางที่เน้นอุปกรณ์เคลื่อนที่เป็นอันดับแรกนั้นถือว่า สมาร์ทโฟน แท็บเล็ต และแอปมือถือเฉพาะงานเป็นเครื่องมือหลักในการทำงานของพนักงาน Bootstrap ตอบสนอง
