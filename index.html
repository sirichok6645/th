ใช้อะไรสร้างเว็บบ้าง

Visual Studio Code

คือ โปรแกรม Code Editor ของ Microsoft ที่ใช้สำหรับเขียน แก้ไขและตรวจสอบความผิดปกติของโค้ด คุณสมบัติของโปรแกรมมีลักษณะเบา มีประสิทธิภาพสูง

HTML

ภาษาในการเขียนและจัดโครงสร้างเนื้อหาของเว็บไซต์ ซึ่งใช้ในการระบุและจัดการองค์ประกอบต่างๆ เช่น ข้อความ รูปภาพ ลิงก์ ฟอร์ม และอื่นๆ ภายในเว็บไซต์

CSS

คือ ภาษาในการกำหนดและตกแต่งรูปลักษณ์ของเอกสาร HTML โดยช่วยให้คุณสามารถปรับแต่งการแสดงผลของเนื้อหาบนเว็บไซต์ เช่น สี ฟอนต์ ขนาด ตัวอักษร การจัดวาง และการจัดเรียงต่างๆ
